## üí° Sample Queries

### Global Summary

```sql
SELECT 
    MAX(date) AS latest_date,
    SUM(daily_cases) AS total_cases,
    SUM(daily_deaths) AS total_deaths
FROM covid_cases;
```


### Top 10 Countries

```sql
SELECT country, MAX(cumulative_cases) AS total_cases
FROM covid_cases
GROUP BY country
ORDER BY total_cases DESC
LIMIT 10;
```


### Hospital Capacity

```sql
SELECT state, AVG(hospital_admissions) AS avg_admissions
FROM hospital_data
WHERE country = 'India' AND date >= '2024-01-01'
GROUP BY state
ORDER BY avg_admissions DESC;
```
